# Simpler (Repository Root)

这个仓库包含三部分：
- `simpler/`：主程序源码（启动器本体）
- `Simplerscripts/`：脚本库（Library，用于挑选/维护脚本）
- `.github/workflows/`：自动打包与发布流程

如果你是使用者，请前往 GitHub Releases 下载打包好的版本。
如果你是开发者，请进入 `simpler/` 查看源码与说明。
